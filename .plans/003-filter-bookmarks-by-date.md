# 日付引数でブックマーク件数を絞る

## 目的

引数で日付を指定した場合、その日付以降に作成された
ブックマークのみを取得して表示できるようにする。

## 仕様案

- 引数なしの場合は従来どおり全ブックマークを取得する。
- `--from yyyyMMdd` を指定した場合は、その値を日付として解釈する。
- `--from` の値が不正な日付、または余分な位置引数がある場合は
  エラーを返す。
- 指定日以降は指定日を含めるため、Raindrop の検索構文
  `created:>YYYY-MM-DD` を使い、指定日の前日を `search` クエリに
  渡して絞り込む。
- API に渡す日付は `yyyyMMdd` から `YYYY-MM-DD` へ変換する。
- 表示形式と並び順は既存仕様を維持する。
